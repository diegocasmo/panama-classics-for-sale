["Panam치 Cl치sicos"](http://panama-clasicos.rhcloud.com/) is a simple WEB scrapper useful for searching classic cars for sale in different countries.

# Installation

## API
- `cd` into the `api/` directory
- Run `npm install` to install dependencies
- Run `node server.js`
- You can now access the API at `http://localhost:8080/`

## Frontend
- `cd` into the `frontend/` directory
- Run `npm install && bower install` to install dependencies
- Run `grunt serve`
- You can now access the frontend application at `http://localhost:9000/`
